# 类型概览

## Line

| 属性名 | 属性含义           | 示例            |
| ------ | ------------------ | --------------- |
| color  | 线条颜色           | #aaa            |
| thick  | 线条粗细           | 25              |
| box    | 线条起点和终点位置 | ((0,0),(25,25)) |


## Image

| 属性名 | 属性含义                   | 示例             |
| ------ | -------------------------- | ---------------- |
| url    | 图片网络地址               | `https://test.com/1.png` |
| box    | 图片左上角位置和右下角位置 | ((0,0),(25,25))  |

## Text

| 属性名      | 属性含义                   | 示例     |
| ----------- | -------------------------- | -------- |
| text        | 详细内容                   | 测试一下 |
| font_size   | 字体大小                   | 15       |
| color       | 字体颜色                   | #aaa     |
| start_point | 文字开始位置               | (0,0)    |
| width       | 文字块宽度（超过自动换行） | 300      |
